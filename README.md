## 🎯 Overview

An interactive visualization and exploration tool for the **[12-Factor Agents](https://github.com/humanlayer/12-factor-agents)** principles - the definitive guide to building reliable LLM-powered applications.

## 🚀 What are 12-Factor Agents?

In the spirit of [12-Factor Apps](https://12factor.net/), **12-Factor Agents** defines core engineering techniques that make LLM-powered software more reliable, scalable, and easier to maintain.

## 📚 The 12 Factors

1. **Natural Language to Tool Calls** - Transform intent into structured actions
2. **Own Your Prompts** - Maintain control over prompt templates
3. **Own Your Context Window** - Actively manage context content
4. **Tools are Just Structured Outputs** - Treat tools as structured JSON
5. **Unify Execution State and Business State** - Single source of truth
6. **Launch/Pause/Resume with Simple APIs** - Support long-running operations
7. **Contact Humans with Tool Calls** - Human-in-the-loop as a tool
8. **Own Your Control Flow** - Explicit flow management
9. **Compact Errors into Context Window** - Intelligent error compression
10. **Small, Focused Agents** - Specialized over general-purpose
11. **Trigger from Anywhere** - Multi-source event handling
12. **Make Your Agent a Stateless Reducer** - Functional agent design
