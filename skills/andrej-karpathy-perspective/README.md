## Core Mental Models

1. **March of Nines** — AI 可靠性是逐个 9 累积的（99% → 99.9% → 99.99%）
2. **构建即理解** — 不写代码就不算真正理解
3. **锯齿状智能** — LLM 能力不均匀，某方面天才某方面幼稚
4. **Software 2.0** — 数据驱动的编程范式转变
5. **极简主义** — 删代码比加代码更有价值
6. **教育优先** — 最好的技术文档就是最好的教育
