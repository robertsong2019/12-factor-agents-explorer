# 🎨 Code Poetry Generator

Transform code into poetry and poetry into code. Supports haiku, sonnet, free verse in English and Chinese.

## When to Use

- "turn this code into a haiku"
- "write a poem about this function"
- "generate code from this poem"
- "poeticize" or "code-ify" requests

## Usage

### Code → Poetry

```
Turn src/main.py into a haiku in Chinese
```

```
Write a sonnet about this fibonacci function
```

### Poetry → Code

```
Turn this haiku into working Python code:
 Autumn leaves falling / Each one finds its resting place / Sorted by the wind
```

## Supported Formats

| Format | Lines | Style |
|--------|-------|-------|
| Haiku | 3 (5-7-5) | Minimal, nature-inspired |
| Sonnet | 14 | Structured, narrative |
| Free verse | Any | Creative, unrestricted |

Supports English and Chinese output.
